# Form and FormTester Java Application

This repository contains a simple Java application that takes user input for their name, last name, and age, and then performs an age check.

## Overview

The application consists of two classes:

*   `Form`: Represents a user's information, including their first name, last name, age, and favorite color.  It also includes a method to check if the user is of legal age.
*   `FormTester`: Contains the `main` method, which handles user input and interacts with the `Form` class.

## Functionality

1.  The `FormTester` class prompts the user to enter their first name, last name, and age.
2.  It creates a `Form` object using the user's input.
3.  It calls the `ageCheck()` method on the `Form` object to determine if the user is of legal age (18 or older).
4.  Finally, it prints a greeting message including the user's name and age.
